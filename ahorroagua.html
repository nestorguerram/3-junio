Rain and Water Level Detection Using Arduino

This document explores the design and implementation of a rain detection and water level monitoring system using Arduino microcontrollers. It includes theoretical foundations, hardware components, wiring diagrams, example code, and external references to videos and tutorials for further learning.

1. Rain (Raindrop) Detection Sensor

Rain detection sensors use a conductive grid that behaves as a variable resistor when water bridges its tracks. It offers both analog and digital output to an Arduino.

Key references:
Circuit DIY
Arduino Get Started

2. Wiring with Arduino UNO

Connect AO to A0, DO to a digital pin (e.g., D4), VCC to 5V, and GND to ground. Use digital pin power control to minimize corrosion.

3. YouTube Tutorials

  • Rain Sensor Tutorial

4. Sample Code

#define POWER_PIN 3
#define DO_PIN 4

void setup() {
  Serial.begin(9600);
  pinMode(POWER_PIN, OUTPUT);
  pinMode(DO_PIN, INPUT);
}

void loop() {
  digitalWrite(POWER_PIN, HIGH);
  delay(10);
  int rain_state = digitalRead(DO_PIN);
  digitalWrite(POWER_PIN, LOW);

  if (rain_state == HIGH)
    Serial.println("No rain detected");
  else
    Serial.println("Rain detected!");
  delay(1000);
}

5. Water Level Detection Sensor

Water level sensors use a conductive strip or floating switches. They provide analog output that increases with water depth.
